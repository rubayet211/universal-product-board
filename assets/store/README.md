This folder contains repository artifacts for the Chrome Web Store listing.

- `screenshot-01-popup-save-flow.png`
  - Based on the popup save flow
  - Shows the extension opened on a shopping page with a product preview and local-first disclosure
  - Exported at `1280x800`
- `screenshot-02-side-panel-board.png`
  - Based on the side panel board

  - Exported at `1280x800`
- `screenshot-03-settings-privacy.png`
  - Based on the Settings page
  - Shows notification, privacy, and import/export controls
  - Exported at `1280x800`
- `promo-small-440x280.png`
  - Small promotional image for the Chrome Web Store listing
  - Focuses on the core “save and organize products” value proposition
  - Exported at `440x280`

## Usage Notes

- These images should stay truthful to the real extension UI and feature set.
- Do not use them to imply price alerts, cloud sync, analytics, or background tracking.
- Replace any placeholder support/contact text in the store listing separately; it is not shown in these images.
